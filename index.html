What is AutoScale?

AutoScale is a Rhino tool for precision scaling based on layer names, with automatic dimension lines, bounding boxes, and annotations.

  • Scale selected geometry based on layers like canvas, seams zips, and annotations
  • Automatically add a reference line and scale label
  • Move scaled geometry to new layers like scale canvas
  • Includes license enforcement and grouping options

Installation

  1. Download and unzip the AutoScale package
  2. Run install.bat to install it to Rhino’s scripts folder
  3. Open Rhino and load the JB Tools.rui toolbar
  4. Click the AutoScale button, or run:
    _RunPythonScript "AutoScale_Run.py"
