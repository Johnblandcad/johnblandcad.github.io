Welcome to AutoScale

AutoScale is a Rhino tool for duplicating, scaling, and annotating geometry across custom layer workflows. Built for marine, textile, and composite production use cases.

⬇ View Downloads

Includes a 30-day free trial • Works with Rhino 7 & 8

Key Features

  • Layer-based scaling with precision control
  • Automatic dimensioning and annotation
  • Centered "SCALED x.xxx" text
  • Group geometry and move to prefixed layers
